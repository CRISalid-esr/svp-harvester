########
Quick start: deployment
########
Welcome to the quick start guide for deployment. Below are the available deployment options:

- **VM deployment:** Set up a new deployment from scratch.
- **Docker Image Deployment:** Deploy using pre-built Docker images.
- **Docker Compose Deployment:** Deploy using Docker Compose for multi-container applications.
- **Kubernetes Deployment:** Deploy using Kubernetes for container orchestration.
